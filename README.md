# GroMath

GroMath is a programming language made for *Crytography* and *Crytanalysis* to hopefully be :
- Cool syntax like SageMath
- As fast as PARI/GP
- As robust as Magma

Without being :
- Slow and spaghetti code like SageMarh
- Horrible syntax like PARI/GP
- $$$$ ... Magma
